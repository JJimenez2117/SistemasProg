Vsalida = Ventrada (1 + (r3/r2))
5v = 1.75v (1 + (r3/220))
5v = 1.75v + 1.75v(r3/220)
5v - 1.75v = 1.75v(r3/220)
3.25v = 1.75v(r3/220)
(3.25/1.75) = (r3/220)
1.857 = (r3/220)
1.857 X 220 = r3

r3 = 408
